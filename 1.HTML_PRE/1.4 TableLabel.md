# TableLabel

### 表格标签
```html
thead为表头
在tr中包含的th为表头行中的不同列
tbody为表内容
在tr中包含的td为内容行中的不同列
border: 表格边框
<table border="1">
    <thead>
        <tr> <th>Index</th> <th>Name</th> </tr>
    </thead>
    <tbody>
        <tr> <td>1</td> <td>Jone1</td> </tr>
        <tr> <td>2</td> <td>Jone2</td> </tr>
        <tr> <td>3</td> <td>Jone3</td> </tr>
        <tr> <td>4</td> <td>Jone4</td> </tr>
    </tbody>
</table>
```
